# QBStudy

## Setup
1. After downloading this repository, download tossups.json from https://drive.google.com/drive/folders/1ZcZ-jnWKULiQ1Sy51as2IMPrhe-5qevL
2. Put tossups.json into the QBStudy folder.
3. Run python -m http.server inside of the project folder.
4. Access the web app using localhost (localhost:8000).
